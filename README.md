Instalasi & Cara Penggunaan

Clone repositori

git clone https://github.com/shandysuryasumarna123-cell/shandy.git  
cd shandy  


Instal dependensi

npm install  
# atau  
composer install  


Konfigurasi lingkungan

Salin .env.example ke .env

Atur variabel seperti DB_HOST, DB_USER, DB_PASS

Jalankan aplikasi

npm run dev  
# atau sesuai instruksi proyek  


Akses di browser: http://localhost:3000 (atau port yang digunakan)
